Detailed Description

3D element composed of 8 points 

 Node numbering (NEKTON symmetric notation)

          3+-----+4    ^ s
          /     /|     |
         /     / |     |
       7+-----+8 +2    +----> r
        |     | /     /
        |     |/     /
       5+-----+6    t
