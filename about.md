# About Neko

Neko is a portable framework for high-order spectral element flow simulations.
It targets modern heterogeneous HPC systems while keeping the codebase suitable
for advanced CFD research and solver development.

The live `neko.cfd` site already exposes releases and documentation, but the
project deserves a stronger presentation layer with:

- a clearer product narrative
- richer visual examples
- a proper publications section
- a separation between landing pages and generated technical documentation

This local prototype is the beginning of that structure.
