# Appendix {#appendix}

## Environmental variable reference {#env-var}

Name                    | Description                                                   | Default value
----                    | -----------                                                   | -------------
`NEKO_AUTOTUNE`         | Force Ax auto-tuning strategy (``'1D'``,``'KSTEP'``)          | Unset
`NEKO_LOG_LEVEL`        | Log verbosity level (integer > 0, default: 1)                 | Unset
